# 图片切割器

## Intro

可以把一张大图切成几个小图。

## Features

1. 多种模式选择，非侵入式添加新模式。

   仅通过为`strategies`一个对象添加新的`key-value`来实现新模式的添加。此处无耦合逻辑。

2. 自动截取图片中间部分，不留白。

   通过我写的一大坨逻辑实现。

3. 使用Vue3 + TypeScript编写，应该挺好。
   
   然而有些地方，尤其是生成DOM的时候，我觉得React写着更好，罢了。


## Usage

选好参数直接下载切割后的图片即可。


## Conclusion

就这样，因为PS技术不好扣不出这种效果所以用JS实现了。
