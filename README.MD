# 图片切割器

## Intro

可以把一张大图切成几个小图。

## Features

1. 多种模式选择，非侵入式添加新模式。

   仅通过为`strategies`一个对象添加新的`key-value`来实现新模式的添加。此处无耦合逻辑。

2. 自动按照比例截取图片中间部分，不留白。

   通过我写的一大坨逻辑实现。

3. 基于`Vue3` + `TS` + `Pinia`，基本都是小型项目配置，没用UI库。

   用来练手挺好的。


## Usage

选好参数直接下载切割后的图片即可。


## Conclusion

就这样，因为PS技术不好扣不出这种效果所以用JS实现了。
